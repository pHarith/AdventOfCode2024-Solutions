# AdventOfCode2024-Solutions
This repo contains my personal solutions to Advent of Code 2024. I will update it regularly as I complete it.

Each numbered folder contains a solution to the problem of that day of the challenge, along with text files containing the problem, the provided input and test inputs. Certain days, whose problem I find more challenge, will have an textfile or even a pdf, where I write an explanation or draw a diagram to explain my own solution, to my future self or anyone viewing the repo.
